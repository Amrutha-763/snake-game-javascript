# Snake Game (JavaScript Web Version)
This is a classic Snake Game developed using HTML, CSS (Grid), and JavaScript. The game runs in the browser and allows users to control the snake using arrow keys.
# Features
- Smooth snake movement using JavaScript game loop
- Keyboard controls (Arrow keys)
- Food generation at random grid positions
- Collision detection (Wall & Self)
- Sound effects integration
- CSS Grid-based game board
# Technologies Used
- HTML5
- CSS3 (Grid Layout)
- JavaScript (DOM Manipulation & Game Logic)
# How to Run
1. Download the project files
2. Open snake.html (or index.html) in browser
3. Use arrow keys to control the snake
